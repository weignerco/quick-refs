# Quick Reference Guides

A collection of deployment guides, setup scripts, and quick reference materials for cloud infrastructure, development tools, and technologies.

## Directory Structure

### `/init-scripts`
Initialization and setup scripts for new environments
- **new-mac-setup.sh** - Initial Mac development environment setup
- **create-oci-vm.sh** - Oracle Cloud VMs provisioning
- **create-oci-vcn.sh** - Oracle Cloud Virtual Cloud Network setup
- **cloud-init.sh** - Cloud initialization scripts

### `/setup-guides`
Step-by-step deployment and configuration guides
- **a10-gpu-test.sh** - A10 GPU deployment and testing guide

### `/cheatsheet`
Quick reference materials and command checklists
- **docker-cheatsheet** - Docker commands quick reference
- **pl-sql-first-queries.sql** - PL/SQL query examples
- **postgresql.sh** - PostgreSQL setup and commands
- **ssh-proxying.sh** - SSH proxy configuration reference
- **streamlit.sh** - Streamlit commands and setup

### Root Level Quick Starts
Beginner-friendly learning materials
- **basics-git.sh** - Git commands for beginners
- **basics-python.py** - Python basics examples
- **basics-markdown.md** - Markdown syntax guide

## Usage

Browse the appropriate subdirectory based on your need:
- Setting up a new environment? → Check `/init-scripts`
- Need deployment instructions? → Check `/setup-guides`
- Need a quick command? → Check `/cheatsheet` or root level basics
